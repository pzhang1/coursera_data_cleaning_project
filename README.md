# coursera_data_cleaning_project
This is the getting and cleaning data course project repository. 
The *run_analysis.R* code did the following:
  - Download the dataset from https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip
  - Load the required information about features, training and testing data sets.
  - Keep only the mean and standard deviation of each measurement.
  - For both the training and testing data, merge the activity and subject data sets, and convert the activity and subject into factors. 
  - Create a tidy data sets that includes the mean of each variable for the activity and subject pair.
